File Mover App

This application is a desktop tool that helps you automatically organize, copy, or move selected files based on your Google Calendar events.

Main Features

  • Fetch events from your Google Calendar
  • Automatically organize and move local files based on event names
  • Use token.pickle for credential persistence
  • User-friendly graphical interface built with CustomTkinter

Google OAuth Scope Used

https://www.googleapis.com/auth/calendar.readonly

About This Application

This application is distributed as a Windows executable (.exe) file and is intended to be used as a local desktop utility. It is not a web application. All OAuth 2.0 authentication flows are initiated locally, and the user's default browser is used during the sign-in process.

The application accesses Google Calendar data only to identify events used to classify and move local files on the user's system. No data is transmitted or stored externally, and all operations take place locally on the user's device.

Please refer to the demo video and screenshots above to understand how the application works in its actual runtime environment.
