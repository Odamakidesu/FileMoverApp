File Mover App

This application is a desktop tool that helps you automatically organize, copy, or move selected files based on your Google Calendar events.

Main Features

  • Fetch events from your Google Calendar
  • Automatically organize and move local files based on event names
  • Use token.pickle for credential persistence
  • User-friendly graphical interface built with CustomTkinter

Google OAuth Scope Used

https://www.googleapis.com/auth/calendar.readonly
